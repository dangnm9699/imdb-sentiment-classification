# imdb-sentiment-classification

Web Mining / IT4868 / SoICT / HUST

**Prerequisites**

- Python 3.8.x
- Ubuntu 20.04

## 1. Setup environtment

- Create venv

```bash
python -m venv .venv
```

- Activate venv

```bash
source ./.venv/bin/activate
```

- Install packages

```bash
pip install -r requirements.txt
```

## 2. How to run

```bash
python imdb_cnn.py
```
